Theme Tutorials

Overview

This tutorial will guide you to setup the theme similar with our demo.

Please take the time to read through the tutorials and take notice of the instructions listed here, as many support related questions can be answered simply by re-reading these tutorials.

Before starting, please install and activate all recommended plugins.

Theme Installation

  1. Unzip the zip file that you download from Wrap Bootstrap and locate the 'hubblewp' folders inside the 'theme' folder. This is the folder and contents you will need to upload to your webserver.
  2. FTP Upload the 'hubblewp' folders to your 'wp-content/themes/' directory on your web server, then you can activate the theme.
  3. After you activate the theme, you can change the theme settings. Login to wordpress admin panel and click the ' Appearance' tab then click ' Theme Options' sub tab .

If you want quick setup for your website, you can import our xml data sample that included in the "import" folder, here are the steps how to import sample data :

  1. Click 'Tools >> Import' then choose 'Wordpress'.
  2. Then browse the xml sample data file.
  3. Click 'Upload file and import'

 

1. Home Page

Tutorial to add the Home page.

  1. Click the 'Pages' tab.
  2. Click the 'Add New' Sub Tab
  3. You can name it 'Homepage' or 'Welcome' or any name that you want.
  4. Select the 'HOME' template in the right 'Page Attribute' section.
  5. There's no need to add content to the home page. All content in the home page will be added in the options panel later in the tutorial.
  6. When you are ready, click Publish.



    Set this home page as front page
  7. In the left side menu, click the 'Settings' tab.
  8. Click the 'Reading' sub tab.
  9. Then change the "Front page displays" to "A static page" and choose the "Welcome" page that you just created.
  10. Save the settings.

    Home page slider
  11. Click 'Options' in the left side navigation menu.
  12. You are now in the options panel. Click on "Home Page".


  13. In this panel you can upload 3 images to the slider. Images for the slider must be 1088px wide by 340px high.
  14. You can add a link to a slide by typing in a link in the Home Page Slider Link text box. Make sure to add the full url, ex: http://www.google.com.
  15. When you are ready, click Save Options.

    Home page content
  16. If you scroll down past the slides you will see an area for the featured home page content.
  17. To upload an image you need to click the "upload" button.
  18. Fill in the necessary information for the title and content.

  19. When you are all done with your home page content and slides please click the "save options" button at the top right or bottom right of the page.

2. Portfolio Page

Tutorial to add Portfolio page

IMPORTANT: In order for your portfolio pieces to display properly you need to make the images the same size.

Add Portfolio Category

  1. On the left navigation there is a link to "portfolio". Click that link.


  2. Input your Project name in the "title" field.
  3. On the bottom right of the page you will see a "featured image" link. Click that link.



  4. Click on "Select files" and select an image for your portfolio.



  5. Select image to upload.
  6. After upload you will see the image information.
  7. Click the 'Use as featured image' to make the image as a primary image.(this primary image will be used as a thumbnail)
  8. You are done, close the upload window.
  9. Input your content.
  10. You may want to add an image for your portfolio post at this time.
  11. You can manage and upload the images with clicking 'Manage Images'.



  12. You will see the wordpress media uploader window show up.
  13. Add an image like you did for the featured post but instead of selecting "use as featured image" you will want to choose "insert into post".
  14. When you are ready, click Publish.
  15. You can add as many portfolio pieces as you want.

    Add Portfolio Page
  16. Click the 'Pages' tab.
  17. Click the 'Add New' Sub Tab
  18. Enter the title 'Portfolio' in the title box.
  19. Select the 'Portfolio' template in the right 'Page Attribute' section.
  20. When you are ready, click Publish.

3. Blog Page

Tutorial to add Blog page

Wordpress.org does a much better job of explaining the details of posting. If you have any questions about posting blogs please refer to this page.
http://codex.wordpress.org/Writing_Posts

Add the Blog Category

  1. Click the 'Post' tab.
  2. Click the 'Categories' Sub Tab.
  3. Click the 'Add New Category' button and you are done!
  4. You will see the category you just added in the right side area.

    Adding a blog post
  5. Click the 'Post' tab.
  6. Click the 'Add New' Sub Tab
  7. Start filling in the blanks.
  8. 'Upload' and 'set featured image' for slider. This featured image will be used for blog thumbnail.
  9. When you are ready, click Publish.
  10. You can add as many blog posts as needed.

    Add the Blog page
  11. Click the 'Pages' tab.
  12. Click the 'Add New' Sub Tab
  13. Enter the title 'Blog' in the title box.
  14. Select the 'Blog' template in the right 'Page Attribute' section.
  15. Just leave the 'content editor box' blank.
  16. When you are ready, click Publish.

    Sidebar
  17. Click 'Appearance >> Widgets'.
  18. Drag any widget to the 'Post Sidebar' position

4. Setup Pricing Tables

Tutorial for setting up the pricing tables.

  1. Add the pricing tables page
  2. Click the 'Pages' tab.
  3. Click the 'Add New' Sub Tab
  4. Enter the title 'Pricing' in the title box.
  5. Select the 'Pricing' template in the right 'Page Attribute' section.
  6. Just leave the 'content editor box' blank.
  7. When you are ready, click Publish.

    Adding pricing tables
  8. Click the ' Options ' tab.
  9. Fill in the information for your pricing. There are examples on the right of the page.

  10. Add a 'Featured' pricing table.
  11. Above each pricing tables options you will see a drop down box for "not featured". Switch it to "featured" to feature a pricing table. You should only featured one pricing table.



5. Setup Contact Page

Tutorial for setting up the Contact Page.

IMPORTANT: All emails sent from your contact page will go to the admins email account, so make sure you have a valid email account in your admin account.

  1. Add the pricing tables page
  2. Click the 'Pages' tab.
  3. Click the 'Add New' Sub Tab
  4. Enter the title 'Contact' in the title box.
  5. Select the 'Contact' template in the right 'Page Attribute' section.
  6. Just leave the 'content editor box' blank if you only want a contact form. You can add any content you'd like though.
  7. When you are ready, click Publish.

 

6. Setup Menus

7. Footer Widget

Tutorial to add Footer widget. First of all, please go to APPEARANCE >> WIDGETS

  1. Add Footer1 Sidebar
    • Add a "text" widget
    • Put the title as Contact
    • Add your contact information.
    • Click "Save"
  2. Add Footer2 Sidebar
    • Add a "latest posts" widget
    • and put the title with : Latest Post
    • click "Save"

  3. Add Footer3 Sidebar
    • drag "search" Widget into Footer3 Sidebar.
    • Input the title with : "Search".
    • click "Save".
  4. Add Footer4 Sidebar
    • drag "pages" Widget into Footer3 Sidebar.
    • Input the title with : "pages".
    • click "Save".
